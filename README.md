# lua_distro_update

Update and clean-up your Linux distro.

## Prerequisites

Need to install `Lua`
[Download Lua](https://www.lua.org/download.html)

## Usage

```bash
curl https://raw.githubusercontent.com/k1lgor/lua_distro_update/main/update.lua \
-o ~/update.lua && \
chmod +x ~/update.lua && \
sudo ~/update.lua
```
